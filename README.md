# MewnBase Workspace

This workspace is organized for decompiled-source modding and packaging.

Quick Map
- basegame/: original game files (untouched). The canonical jar is basegame/game/desktop-1.0.jar
- src/: decompiled Java sources (your edits go here)
- out/: compiled class output
- work/: scratch area for packaging (not used for compilation)
- tools/: scripts (autoupdate, check_status, difffind)
- backups/: auto backups
- old/: archived copies and unused variants
